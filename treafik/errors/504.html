The Upstream Server Took Too Long

Our gateway waited patiently… and then eventually gave up.

The server acting as a gateway or proxy did not receive a timely response from an upstream server. Imagine OctoJoom tapping its many feet, staring at a loading spinner that never finishes. At some point, it decides enough is enough and returns this timeout message.

What might help:

  • Try again from the homepage or a different section.
